Detailed Description

Codice per le applicazioni FreeRTOS e la gestione dei task di sistema.

Questo file contiene l'inizializzazione del kernel FreeRTOS, la creazione di mutex, thread e la definizione della logica di esecuzione dei task per il controllo, la comunicazione e la diagnostica del sistema.
