Detailed Description

Interfaccia di basso livello per la lettura degli encoder hardware via timer.

Gestisce il calcolo del delta dei ticks, risolvendo le problematiche di overflow dei timer a 16 bit e verificando la plausibilità fisica delle letture.

Function Documentation

◆ Encoder_ReadDelta()

EncoderStatus_t Encoder_ReadDelta ( uint8_t encoder_id,
int * delta_ticks )

Legge la variazione (delta) dei ticks dall'ultima chiamata per un dato encoder.

Legge la variazione di tick (delta) con gestione dell'overflow.

Calcola la differenza gestendo l'overflow del timer circolare e verifica che il movimento non superi i limiti fisici definiti da MAX_DELTA_TICKS.

Parameters
[in]encoder_idIndice dell'encoder (0 a 3).
[out]delta_ticksPuntatore alla variabile dove scrivere il delta calcolato.
Returns
EncoderStatus_t OK se la lettura è valida, OVERFLOW se supera i limiti fisici.
